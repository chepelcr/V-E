# Development Prompt: Dynamic Construction & Real Estate Website (JSON-Driven)


## Assigned Role
Act as a **Senior Full-Stack Developer and Software Architect**. Your goal is to design and implement the architecture for a lightweight web application for a **Construction Company**, based on **React (SPA) with TypeScript** that dynamically consumes data from JSON files, along with an administration panel (Dashboard) that will run locally in its first version. The architecture must prioritize minimal execution/hosting costs (Jamstack or static hosting) and complete modularity. **pnpm** must be used as the package manager.


---


## 1. Context and Architecture Rules
* **Content Delivery (JSON-Driven):** The frontend must act as an abstract rendering engine. No text, image paths, prices, or geographic configurations should be hardcoded. Everything must be consumed from JSON files.
* **Tech Stack Constraints:** Use React with **TypeScript** (Strict Mode) to ensure type safety for all JSON data structures. Use **pnpm** for fast, disk-space efficient package management.
* **Low Operational Cost:** The application must be lightweight, optimized for deployment on low-cost or free static hosting services (e.g., Netlify, Vercel, GitHub Pages) that read local JSON data.
* **Local Dashboard (V1):** Design a complementary admin panel to run locally (e.g., an interface integrated into the app protected by a local route, or a local Node/Vite environment) that allows reading, modifying, and saving configurations back to the JSON files.
* **Absolute Modularity:** All text blocks, menus, images, and catalogs must be fully editable from the JSON.


---


## 2. Design Guidelines (UI/UX)
* **Background/Theme:** Premium background emulating a black marble wall with gold veins (subtle texture via CSS or optimized image to avoid performance drops).
* **Brand Identity:** Prominent and elegant positioning for the **Company Logo**. The secondary color palette and typography must contrast perfectly against the dark and gold background (elegant and minimalist style).
